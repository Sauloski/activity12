# ChuckJokesVue

This is a simple Vue.js application that displays a list of Chuck Norris jokes. The project was created as an activity for a web design class.

## Features

*   Displays a predefined list of jokes.
*   Built with Vue.js and Vite.
*   Styled with CSS for a clean and simple look.

## How to run the project

1.  Install the dependencies:
    ```bash
    npm install
    ```
2.  Run the development server:
    ```bash
    npm run dev
    ```
